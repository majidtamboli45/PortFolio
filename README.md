# Personal Portfolio Website

This repository contains my **personal portfolio website** showcasing my profile, skills, projects, and resume. The website is built using **HTML, CSS, and JavaScript** and is hosted using **GitHub Pages**.

## 🌐 Live Website

You can view the live portfolio here:
👉 https://majidtamboli45.github.io/PortFolio/

## 📌 Features

* Personal introduction section
* Profile image display
* Skills and project showcase
* Downloadable resume
* Responsive design for different devices
* Clean and simple user interface

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **JavaScript** – Interactive functionality
* **GitHub Pages** – Deployment

## 📂 Project Structure

```
PortFolio/
│
├── index.html      # Main webpage
├── style.css       # Styling of the website
├── script.js       # JavaScript functionality
├── profile.jpg     # Profile image
├── resume.pdf      # Downloadable resume
└── README.md       # Project documentation
```

## 🚀 How to Run the Project

1. Clone the repository

```
git clone https://github.com/majidtamboli45/PortFolio.git
```

2. Open the folder

```
cd PortFolio
```

3. Open `index.html` in your browser.
